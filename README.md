# omu-covid19-chart(開発中)
## 概要
* 大阪公立大学の新型コロナウイルス感染症の発生状況(検査結果判明日)をグラフ化したものです。</p>
* データは2022年8月5日時点のものを使用しています。</p>
* 最新の情報は[大学ホームページ](https://www.omu.ac.jp/about/efforts/covid19/)をご覧ください。</p>
## 注意
* データ処理にあたり、検査結果判明日が不明とされているものは除外してあります。
